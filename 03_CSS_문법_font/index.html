폰트의 서체 적용

1. 외부에서 서체를 가져온다. google fonts라는 사이트로부터 링크 연동을 진행

영문과 한글 혼용시, font-family: '영문서체명', '한글서체명', serif; ==> 영문서체는 한글영역에서 표현못함 / 한글서체는 영문과 한글 모두 표현 가능

2. 자체 서버에 저장된 폰트를 사용하고자 할 때, transfonter.org 사이트로 접속하여 폰트의 원본파일을 변환작업(eot, ttf, woff, woff2, svg 체크박스 모두 체크할 것!)을 시킨 후 다운받아 가져온다. css에서 각 폰트 파일 연동을 선언한다.
