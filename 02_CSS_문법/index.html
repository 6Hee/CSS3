태그(요소명) 선택자 : 각괄호 내부의 첫 선언요소명을 기준으로 스타일 적용시 사용

클래스 선택자 : 태그 내부에 추가로 작성된 클래스(class) 속성과 그 속성이 담고 있는 속성값을 기준으로 선택하여 스타일 적용시 사용

클래스 선택지 선언시 주의사항

1. 클래스명을 선언시 첫글자는 숫자를 넣을 수 없음(단, 영문 이후에 나오는 숫자는 상관없음. 예시 box1)

2. 클래스명은 복수의 클래스명 선언이 가능(단, 첫번째 클래스명 작성 이후 반드시 한칸을 띄우고 두번째 클래스 명을 작성해야 함)

3. 클래스명 선언시 반드시 의미를 부여하여 이름을 지어주어야 함

4. 클래스명 내부 단어를 연결시, 사칙연산 기호를 작성하지 말 것! 특수문자 사용하지 말 것! (단 '단어_단어'사용을 추천

5. 클래스명은 재활용 또는 재사용이 가능 (<-> 아이디 선택자)

아이디 선택자 : 태그 내부에 id 라는 속성에 선언된 속성값을 선택하여 요소에 스타일 적용시 사용

아이디 선택자 선언시 주의사항

1. 아이디 선택자 선언시 숫자가 먼저 작성되면 안됨

2. 복수의 아이디명 사용 불가

3. 동일한 아이디명도 사용불가(=> css에서는 스타일 동시 적용이 가능하나, 자바 스크립트에서는 맨 처음 나오는 아이디명만 인정

4. 가급적이면 덩치가 큰 영역을 감싸는 공간에 부여하면 추후에 명시도를 올려서 적용하기가 수월해짐

자식 선택자 : 부모와 자식 관계가 확실한 요소에 스타일 적용시 사용
