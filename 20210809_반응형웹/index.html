반응형 웹이란 무엇인가?

각 디바이스별로 특징을 갖고 있는데, 하나의 사이트(소스코드)가 각 디바이스별로 [가로 사이즈]가 조정되었을 때 어떻게 대응을 할 것인가를 고민하여 레이아웃이 조정되는 웹사이트를 일컬음.

@media (max-width:000px){ } : PC view -> Mobile view로 반응형이 진행되는 과정을 구성시 사용 : 큰 가로값부터 작은 가로값으로 미디어쿼리를 설정한다.

@media (min-width:000px){ } : Mobile view -> PC view로 반응형이 진행되는 과정을 구성시 사용 : 작은 가로값부터 큰 가로값으로 미디어쿼리를 설정한다. (중복구간을 피하기 위함)
