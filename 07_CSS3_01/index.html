border-radius : 모서리를 둥글게 처리하는 선언

border-radius : 20px; => 좌상단, 우상단, 우하단, 좌하단의 모든 모서리를 원형의 반지름 값인 20px로 둥글게 처리해라

border-radius : 10px 20px 30px 40px; => 좌상단으로부터 시계방향의 모서리를 각각 적용된 값으로 둥글게 처리해라. border-radius: 좌상단 우상단 우하단 좌하단;

border-radius : 10px 40px => 좌상단(우하단) 우상단(좌하단)

border-radius : 10px 40px 0px => 좌상단 우상단(좌하단) 우하단

1/4 원을 구성할 때 : width값과 height 값은 동일한 선조건

1/2 원을 구성할 때 :width값 또는 Height 값이 상대적으로 두배일 때 가능

타원을 구성할 때 : width 값과 height 값이 동일하지 않을 때 가능 + 50%이상의 라디우스 적용

캡슐형태를 구성할 때 : width값과 height 값이 동일하지 않을 때 가능 + 고정px
