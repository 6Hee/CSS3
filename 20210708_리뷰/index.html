블록과 인라인 요소를 가진 태그 정리

블록 요소를 갖고 있는 태그 : div, h1~h6, p, ul(ol) -> 자식으로 블록 요소와 인라인 요소를 갖고 있는 태그 모두 사용 가능

인라인 요소를 갖고 있는 태그 : a, img, span, (input, textarea, select) -> 자식으로 인라인 요소를 갖고 있는 태그만 사용 가능

범용요소 : li -> 블록 태그와 동일 성질

ul~li를 적용하여 리스트를 만드는 과정에서 개수가 하나만 존재하는 경우가 아니라면 사용

태그 중에서 필수 속성

div : 필수속성 없음

h1~h6 : 필수속성 없음

p : 필수속성 없음

a : href

img : src, alt

span : 필수속성 없음

input, buttom : type이라는 속성은 입력된 또는 선택된 데이터의 값(value)을 전달하기가 수월해짐

select, textarea : 필수속성 없음, name이라는 속성은 입력된 또는 선택된 데이터의 값(value)을 전달하기가 수월해짐
