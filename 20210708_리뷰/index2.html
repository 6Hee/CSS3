스타일 연동 방식

인라인 스타일 시트 : HTML문서 내에 존재하는 태그 내부에 style 속성을 선언하여 작성된 스타일 시트(우선순위가 가장 높음 -> 추후 자바스크립트에서 직접 변경 시켜주는 스타일도 가능)

내부 스타일 시트 : HTML문서 내에 style 태그를 작성하여 지목한 선택자를 기준으로 스타일 선언시 사용

외부 스타일 시트 : HTML 문서 파일과는 별도로 외부에 스타일 시트 파일을 구성한 후 연동하여 지목한 선택자의 스타일 적용 시 사용(관리차원에서 가장 효율적)

선택자 파트

전체 선택자 : *{}

태그(요소명) 선택자 : 태그{}

클래스 선택자 : .클래스명{}

아이디 선택자 : #아이디명{}

자식 선택자 : 부모 > 자식{}

후손 선택자 : (부모 또는 조상) (자식 또는 자손){} (원거리 자손 또는 근거리 자손도 선택 가능)

수도 선택자 : nth-child(n) / first-child / last-child 또는 nth-of-type(n) / first-of-type / last-of-type => 내가 부모로부터 몇번째 자식인가?

속성 선택자 : [속성명^="연속된 글자"] (해당하는 속성이 연속된 글자로 시작하는 것을 선택) / [속성명$="연속된 글자"] (해당하는 속성이 연속된 글자로 종료하는 것을 선택) / [속성명*="연속된 글자"] (해당하는 속성이 연속된 글자가 포함된 것을 선택) /

병렬 선택자 : 선택자1, 선택자2{}(두 가지 이상의 선택자를 동시에 동일한 스타일 적용시 사용 - 주의사항은 마지막에 작성된 선택자 이후에는 콤마, 를 작성 하지 않는다.)

종속 선택자 : div.클래스명{} / .클래스명#아이디명{} / 태그명#아이디명{} / .클래스명1.클래스명2{} (띄어쓰기가 없는 상태에서 하나의 태그 내부에 여러가지 속성에 관한 속성값을 붙여서 사용하기 - #txtdiv(#아이디명태그명)는 제외)
