배경색상 (background-color) - 문자(red,orange 등) 16진수 코드(#ff0000 = #f00) , rgba(255, 0, 0,5), hsb(색상각도(deg), 채도비율(%), 명도비율(%))

배경 이미지 (background-image:url(이미지의 파일 및 경로))

배경의 위치 (background-position : 수평위치, 수직위치)

수평위치 선언 가능한 값 : left, center, right / 0%~100% / px

수직위치 선언 가능한 값 : top, center(middle은 사용불가) bottom / 0%~100%

배경의 크기 (background-size : 가로사이즈 세로사이즈 / cover / contain )

가로사이즈 및 세로사이즈에 적용 가능한 단위 : %, px emd / auto : 가로 또는 세로가 고정으로 지정되었을 때 나머지의 비율 에 의해 맞춰준다.

parallax 효과를 구현할 수 있는 배경 붙이기 (background-attachment : scroll(기본값) / fixed)

통함 background

background : 배경색상, 배경이미지, 반복여부, 배경 위치 배경 사이즈, 배경 붙이기
